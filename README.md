# Custom PC Build Project 💻

## Overview

I built this PC from scratch as a hands-on way to better understand how computer hardware and software come together. Instead of just learning concepts, I wanted to physically go through the process — from selecting parts to troubleshooting real issues.

## The Build Journey

I started by researching and selecting all the components, making sure everything was compatible — CPU, motherboard, RAM, GPU, storage, power supply, and case. Once everything arrived, I began assembling the system piece by piece.

Installing the motherboard, CPU, and RAM went smoothly, but things got more real when I started wiring everything together. Managing cables and making sure every connection was secure took patience, especially with the power supply and front panel connectors.

## Challenges Along the Way

When I powered the system on for the first time, it didn’t boot properly. The system turned on, but I wasn’t getting the expected display output.

I went into troubleshooting mode:

- Rechecked all power connections
- Reseated the RAM and GPU
- Verified monitor and cable connections

After going step by step, I realized one of the components wasn’t fully seated. Once I fixed that, the system successfully booted into BIOS.

## System Setup

After getting into BIOS/UEFI, I:

- Verified that all hardware was recognized
- Adjusted basic settings
- Installed Windows OS
- Installed drivers and ran system updates

Seeing everything come together after the initial issues was honestly the most rewarding part.

## Skills Gained

- Hardware assembly and component compatibility
- BIOS/UEFI navigation and configuration
- Windows OS installation and setup
- Driver installation and updates
- Troubleshooting hardware and boot issues
- Cable management and airflow optimization

## Outcome

I successfully built a fully functional desktop PC and gained real hands-on experience that you can’t get from just watching videos. This project helped me build confidence in diagnosing and solving hardware issues, which is directly relevant to IT support roles.
